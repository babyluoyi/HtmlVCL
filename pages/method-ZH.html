方法

  • ChromeWebBrowser
    语法: ChromeWebBrowser(CSharpBrowserSettings settings)
    参数: settings
    浏览器控件的初始化参数.CSharpBrowserSettings类定义了以下参数:
    CachePath, Locale, LocaleDirPath, UserAgent
    例子: CSharpBrowserSettings settings = new CSharpBrowserSettings();
    settings.UserAgent = "My Browser 1.0.4.0";
    settings.CachePath = @"C:\temp\caches";
    ChromeWebBrowser browser = new ChromeWebBrowser(settings);
    备注: 构造函数.构造函数有两个重载方法,一个是不带settings函数,这种是将不设置浏览器参数。
  • Initialize
    语法: public int Initialize(CSharpBrowserSettings settings)
    参数: settings
    浏览器控件的初始化参数.CSharpBrowserSettings类定义了以下参数::
    CachePath, Locale, LocaleDirPath, UserAgent
    例子: CSharpBrowserSettings settings = new CSharpBrowserSettings();
    settings.UserAgent = "My Browser 1.0.4.0";
    settings.CachePath = @"C:\temp\caches";
    ChromeWebBrowser browser = new ChromeWebBrowser()
    browser.Initialize(settings);
    返回: 0:成功, -1:失败
    备注: 初始化浏览器环境。

  • OpenUrl
    语法: public void OpenUrl(string Url)
    public void OpenUrl(object request)
    参数: Url: 打开指定的网址.
    request：在NewWindow事件中使用.
    例子 1: browser.OpenUrl(newUrl);
    例子 2: private void NewPage(string newUrl,object req)
    {
       TabPage newPage = new TabPage(newUrl);
       tabControl1.TabPages.Add(newPage);
       tabControl1.SelectTab(newPage);
       ChromeWebBrowser browser = new ChromeWebBrowser();
       browser.BrowserNewWindow += new NewWindowEventHandler(chromeWebBrowser1_BrowserNewWindow);
       browser.Initialize();
       newPage.Controls.Add(browser);
       browser.Validate();
       browser.Dock = DockStyle.Fill;
       if (newUrl.Contains("&"))
        browser.OpenUrl(req);
       else
        browser.OpenUrl(newUrl);
    }
    备注: 打开指定网址，打开之前必须调用Initialize函数初始化环境。

    参数: script

    }

    }
    备注: 立即执行javascrip代码，并返回javascript对象

事件

  • BrowserCreated 浏览器控件创建事件
  • BrowserNavigating 浏览器打开前事件
  • BrowserNavigated 浏览器打开后事件
  • BrowserDocumentCompleted 浏览器文档加载完成事件
  • BrowserFrameLoadStart、PageLoadStartEventHandler 某个Frame元素加载页面开始事件
  • BrowserFrameLoadEnd、PageLoadFinishEventhandler 某个Frame元素加载页面结束事件
  • BrowserFrameLoadError 页面加载错误事件
  • BrowserTitleChange 浏览器加载页面时，标题改变事件
  • BrowserUrlChange 浏览器加载页面时，网址改变事件
  • BrowserPreviewKeyDown 浏览器键盘事件
  • BrowserBeforeDownload 文件下载前事件
  • BrowserDownloading 文件下载中事件
  • BrowserNewWindow 新窗口事件

类

  • CwbDocument
